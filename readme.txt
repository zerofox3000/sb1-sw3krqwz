Accept payments and manage transactions with Stratos Pay merchant services.

== Description ==

Stratos Pay provides cutting-edge payment solutions designed to streamline your business operations and enhance customer experience.

= Features =

* Seamless WordPress integration
* Detailed transaction reporting
* Advanced fraud protection
* Competitive transaction rates
* 24/7 customer support

= Requirements =

* WordPress 5.0 or higher
* PHP 7.2 or higher
* Stratos Pay merchant account

== Installation ==

1. Upload the `stratos-pay` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Stratos Pay dashboard page to configure your API key

== Frequently Asked Questions ==

= Do I need a Stratos Pay account to use this plugin? =

Yes, you need to sign up for a Stratos Pay merchant account to obtain your API key and process payments.

= Is this plugin compatible with WooCommerce? =

Yes, this plugin is fully compatible with WooCommerce and supports all standard WooCommerce features.

= How do I get support? =

For support, please contact support@stratospay.com or visit our website.
